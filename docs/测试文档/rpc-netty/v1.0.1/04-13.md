2. 概述

2.1 测试目的

主要检查当前用伪连接池的情况下
- 所有请求预期时间 小于 1s
- 所有的接口返回正确结果

2.2 测试范围

- sayHello 无返回
- selectAll 返回List集合

2.3 测试环境

Windows 8核 32G 、Maven3.X、JUnit
