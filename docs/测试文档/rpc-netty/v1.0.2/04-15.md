2. 概述

2.1 测试目的

主要检查当前用连接池的情况下
- 所有请求预期时间 小于 1s
- 所有的接口返回正确结果
- QPS提升50%以上
- 50000条请求压测，查看稳定性

2.2 测试范围

- sayHello 无返回

2.3 测试环境

Windows 10 、Maven3.X、JUnit
